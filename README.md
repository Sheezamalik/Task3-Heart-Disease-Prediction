# Task 3: Heart Disease Prediction

**Objective:** Predict whether a person is at risk of 
heart disease based on health data.

**Dataset:** Heart Disease UCI Dataset (Kaggle)

**Models Applied:** Logistic Regression, Decision Tree

**Key Results:**
- Dataset has 920 rows and 16 features
- Missing values handled using median imputation
- Logistic Regression outperformed Decision Tree
- Age, thalch and oldpeak are most important features
- Evaluated using Accuracy, ROC-AUC and Confusion Matrix

**Tools Used:** Python, Pandas, Scikit-learn, Seaborn, Matplotlib
